\hypertarget{index_autotoc_md6}{}\doxysection{\texorpdfstring{Estrutura do projeto}{Estrutura do projeto}}\label{index_autotoc_md6}
Este projeto está dividido em duas partes\+:

A primeira visa a implementação de um leitor exibidor capaz de ler uma arquivo ponto class e apresentar as estruturas nele contidas, de forma similar ao visualizador jclasslib (Opção 1 do menu ao exectuar o código). Para isto Foram aplicados os conceitos estudados em sala de aula para desenvolver usa uma estrutura de dados chamada Class\+File para armazenar as informações lidas do arquivo de bytecode.
